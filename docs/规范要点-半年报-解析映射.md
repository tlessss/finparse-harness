# 半年报披露规范 → 解析逻辑映射（对照年报第2号）

> 依据：《公开发行证券的公司信息披露内容与格式准则第3号——半年度报告的内容与格式（2025版）》
> 原文：`llm-docs/第3号-半年度报告的内容与格式-2025版.pdf`（32 页）
> 修订说明：`llm-docs/第3号-半年度报告-修订说明-2025.pdf`
> 官方公告：https://www.csrc.gov.cn/csrc/c101954/c7547590/content.shtml（证监会公告〔2025〕4号，2025年7月1日起施行）
> 对照：`docs/规范要点-解析映射.md`（年报第2号 2025版）
> 用途：判断 FinParseAI 现有字段在半年报中的**强制程度**，避免把年报 FieldSpec 硬套到半年报。

---

## 〇、最重要的 5 个发现

1. **半年报没有与年报第二十五条等价的「收入与成本」细则**
   - 年报第2号 **第二十五条（一）1.** 强制：按行业/产品/地区/销售模式说明营业收入构成，≥10% 分项列示收入/成本/毛利率。
   - 半年报第3号 **第二十三条（一）** 仅要求：披露收入、成本、费用、研发投入、现金流等并**分析变化原因**（散文式，无维度、无 10% 门槛、无毛利率分项）。
   - 对 PDF 全文检索：**「销售模式」「收入与成本」「分行业」「分产品」「分地区」均未出现**。

2. **MD&A 引用的是年报「第二十三条原则」，不是「第二十五条」**
   - 半年报 **第二十一条**：披露 MD&A 应遵守《年度报告准则》**第二十三条所列原则**（可靠性、相关性、关联性等 7 条原则）。
   - 年报 **第二十三条** = 披露原则；**第二十五条** = 主要经营情况细则（含 revenue_breakdown 等）。
   - 网上部分解读称「半年报与年报收入披露保持一致」——**与 2025 版第3号正文不符**，勿采信。

3. **条款编号不可直接对齐**
   - 半年报 **第二十五条** = 董事/高管离任说明（年报同号为 MD&A 主要经营情况）。
   - 半年报 **无** 与年报「第三十四条 员工情况」对应的强制条款；全文无「专业构成」「教育程度」。
   - 半年报 **无** 前五大客户/供应商相关条款（全文无「供应商」「前五」）。

4. **财务指标计算仍引用年报第2号**
   - 半年报 **第十九条**：主要会计数据/指标按《年度报告准则》及证监会其他规范**计算和披露**（含扣非、股份支付后净利润等）。
   - 这是**指标口径**的引用，不等于 MD&A 披露结构与年报第二十五条相同。

5. **实务中仍可能出现类似表格，但属惯例/自愿，非第3号强制**
   - 交易所模板、公司沿用年报格式、投资者习惯，可能导致半年报 MD&A 仍出现「分行业/分产品」表。
   - 解析器应区分：**准则强制字段** vs **有则解析、无则不算漏**；校验规则不能照搬年报硬规则（如占比和=100%）。

## 一、与 FinParseAI 现有字段的对照

| 字段 | 年报第2号（强制条款） | 半年报第3号（2025） | 半年报解析建议 |
|---|---|---|---|
| **revenue_breakdown** | 第二十五条（一）1. 收入与成本：四维度 + 10% 分项 | 第二十三条（一）：仅定性披露收入/成本/费用等并分析变化 | **非强制同结构**；有表则按语义认表，缺失不算违规 |
| **cost_breakdown** | 第二十五条（一）1. 营业成本主要构成项目占比 | 无对应条款（全文无「营业成本」构成细则） | 半年报通常**不强制**；附注可能有，属财务报告节 |
| **rnd_info** | 第二十五条（一）3. 研发投入（人员/总额/资本化） | 第二十三条（一）仅列「研发投入」为讨论项之一 | **大幅弱化**；仅要求分析性披露，无结构化指标清单 |
| **employees** | 第三十四条 员工情况（专业构成/教育程度等） | **无** 员工情况专条 | 半年报**不要求**；现有 employees 字段默认不适用 |
| **top_clients / top_suppliers** | 第二十五条（一）1. 前5名汇总比例 + 鼓励列名 | **无** 客户/供应商专条 | 半年报**不要求**；有则解析，无则合规 |

## 二、半年报 MD&A 结构（2025 版，与年报不同）

| 条款 | 内容 | 解析相关 |
|---|---|---|
| **第二十条** | 行业/业务/产品/经营模式/市场地位/业绩驱动因素；新增非主营业务说明 | 页定位锚点：「公司所属行业」「主要业务」 |
| **第二十一条** | MD&A 遵守年报准则**第二十三条原则** | 原则性，无具体表头 |
| **第二十二条** | 核心竞争力变化 | — |
| **第二十三条** | 主要经营情况（收入/成本/费用/研发/现金流 + 资产负债/投资/出售/子公司/结构化主体） | **半年报 MD&A 核心**，但无 revenue 四维度细则 |
| **第二十四条** | 风险因素 | — |

> 2025 修订说明：MD&A **篇章顺序调整**——先业务/行业（第二十条），再讨论分析（第二十三条），与 2021 版「先财务讨论、后行业业务」不同。找页时勿沿用年报小节标题顺序。

## 三、半年报其他章节（与年报差异摘要）

| 主题 | 半年报条款 | 说明 |
|---|---|---|
| 主要财务指标 | 第十九条 | 引用第2号计算口径；可披露股份支付后净利润 |
| 利润分配 | 第二十六条 | 半年报特有：董事会审议利润分配/转增预案说明 |
| 股权激励 | 第二十七条 | 披露本报告期实施情况 |
| 环境信息 | 第二十八条 | 披露纳入依法披露名单的企业数量/名称及查询索引 |
| 重要事项 | 第三十条–第四十一条 | 承诺、占用资金、担保、审计、诉讼、关联交易等；**无** 员工节 |
| 股份变动 | 第六节 | 2025 版将优先股情况**并入**本节（不再单章） |

## 四、章节定位锚点（半年报）

| 目标 | 准则章节 | 建议锚点 |
|---|---|---|
| 经营讨论（含收入/成本定性分析） | 第三节 MD&A · 第二十三条 | 「主要经营情况」「主要经营业务」 |
| 行业/业务概览 | 第三节 · 第二十条 | 「公司所属行业」「主要业务」「经营模式」 |
| 主要财务指标（表格式） | 第二节 · 第十九条 | 「主要会计数据和财务指标」 |
| 员工 / 前五大客户供应商 | — | **半年报准则无强制位置**；若出现多为附注或自愿披露 |

## 五、对 FinParseAI 的直接修正项（若支持半年报）

1. **`doc_type` / `report_quarter`**：抽象 `annual` | `interim`，勿全链路硬编码 `annual`（见 `api.py`、`console_service.py`、`llm_judge.py` 等）。
2. **FieldSpec 分类型**：`revenue_breakdown` 等应挂 `applicable_doc_types: [annual]` 或半年报专用宽松版（无 10% 门槛、无四维度强制、缺失不判 fail）。
3. **RAG 集合名**：`{code}_{year}_annual` → `{code}_{year}_{quarter}` 或 `{code}_{doc_id}`。
4. **PDF 下载**：`ensure_annual_pdf` → 支持半年报公告类型（book-agent 已有 `interim_report` 分类）。
5. **verify / 硬规则**：半年报 revenue 表若存在，**禁止**套用年报「四维度齐全 + 占比和=100%」作为 pass 条件；应标注 `voluntary_disclosure` 或降低 severity。
6. **字段优先级**：半年报首期可只做 `主要财务指标（第十九条）` + MD&A 第二十三条定性抽取；employees / top_clients 等可标记 N/A。

## 六、常见误读澄清（均对照 2025 版 PDF 原文）

| 误读 | 事实 |
|---|---|
| 「半年报与年报收入构成披露一致」 | 第3号全文无「销售模式」「收入与成本」；无 10% 营收分项门槛 |
| 「半年报第二十五条 = 收入与成本」 | 半年报第二十五条 = **董事/高管离任** |
| 「MD&A 遵守年报第二十五条」 | 半年报第二十一条引用的是年报 **第二十三条（原则）** |
| 「半年报也要披露员工专业构成」 | 第3号**无**员工情况专条；「员工」仅出现在股权激励/员工持股语境 |
| 「半年报也要披露前五大客户供应商」 | 第3号**无**相关条款 |

## 七、值得解析的数据（推荐字段与优先级）

> 判断标准：**准则强制** × **结构化程度** × **时效价值** × **能否复用 Harness**；不重复 Wind/DB 三大报表主表。

### P0 — 首期强烈建议（强制 + 表格式 + 高价值）

| # | 建议字段 | 准则条款 | 典型内容 | 分析价值 | FieldSpec 草案 |
|---|---|---|---|---|---|
| P0-1 | **main_financials** | 第十九条 | 总资产、营收、归母净利润、扣非、净资产、经营现金流、ROE、EPS；报告期 vs **上年同期**及变动率；非经常性损益明细；可选股份支付后净利润 | H1 业绩时效性最强（7–8 月即得） | cls=C/B；锚点「主要会计数据和财务指标」；`applicable: [interim]` |
| P0-2 | **interim_dividend_plan** | 第二十六条 | 半年度利润分配或公积金转增预案；是否符合章程/审议程序 | **半年报独有**；中期分红/送转事件驱动 | 规则+模板；锚点「利润分配预案」「公积金转增」 |
| P0-3 | **subsidiary_analysis** | 第二十三条（六） | 主要子公司：业务、注册资本、总资产、净资产、净利润；取得/处置子公司；单家净利润或参股投资收益占合并 **≥10%** 时须披露主营业务收入/利润 | 集团型公司分部洞察，早于年报 | cls=B；锚点「主要控股参股公司」「主要子公司」 |

### P1 — 第二期建议（强制但半结构化，或附注复用）

| # | 建议字段 | 准则条款 | 说明 |
|---|---|---|---|
| P1-1 | **asset_liability_mix** | 第二十三条（三） | 货币资金/应收/存货/固定资产/借款等占总资产比重及重大变动；有表则抽，纯文字则 N/A |
| P1-2 | **investment_status** | 第二十三条（四） | 金融资产、重大非股权投资、重大股权投资三类；版式不统一，先抓有标准表的子集 |
| P1-3 | **rnd_info**（附注路径） | 第五十三条附注 | MD&A 仅定性；**附注「研发费用」科目表**结构与年报相近，复用现有 FieldSpec，改 section 锚点至附注 |
| P1-4 | **revenue_breakdown**（宽松） | —（自愿） | 约 30–50% 公司仍披露类似表；有则按语义认表，**无则 pass**；verify 标 `voluntary_disclosure`，禁止四维度+占比和硬规则 |
| P1-5 | **equity_incentive** | 第二十七条 | 股权激励/员工持股本报告期实施情况；模板化中等 |

### P2 — 可选（事件类，偏 caibaoxia / 公告流水线）

| 数据 | 条款 | 说明 |
|---|---|---|
| 重大关联交易 | 第三十八条 | 累计 ≥3000 万且占净资产 5%（科创板另有标准） |
| 重大合同 | 第三十九条 | 损益达利润总额 10% |
| 对外担保 / 资金占用 | 第三十一–三十二条 | 风险事件推送 |
| 承诺事项 | 第三十条 | 履约跟踪 |
| 股东持股 / 户数 | 第四十三条 | Wind 常有，低优先级 |
| 风险因素 | 第二十四条 | 散文，LLM 切片，非 FinParse 勾稽主战场 |

### 不建议作为半年报目标

| 字段/数据 | 原因 |
|---|---|
| **employees** | 无员工情况专条 |
| **top_clients / top_suppliers** | 无专条，实务极少单独披露 |
| **cost_breakdown** | 无 MD&A 强制细则 |
| **三大报表主表** | DB/Wind 已有 |
| **MD&A 散文**（行业/竞争力/风险） | 价值在解读不在勾稽 → caibaoxia |

### 实施路线图

```
Phase 0（最小可用）
├── main_financials
├── interim_dividend_plan
└── doc_type=interim 全链路（PDF / DB / RAG 命名）

Phase 1（复用 Harness）
├── subsidiary_analysis
├── rnd_info（附注路径）
└── revenue_breakdown（宽松模式 + voluntary 标记）

Phase 2（对齐 docs/第二期字段规划.md P1）
└── 附注：应收账龄 / 存货 / 关联交易；重要事项事件推送
```
